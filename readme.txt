This is a program made to track the effectiveness of resonant breathing. This is breathing at certain prolonged intervals of inhalation and exhalation, with the goal of reducing heart rate to create calmness. The program records data you submit concerning the number of heartbeats you have over roughly two minutes while following the advised breathing intervals. With several data points, you can analyze the data to determine the optimatl breathing itnervals to lower the heart rate the most.
